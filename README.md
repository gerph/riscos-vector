# *Vector tool

The *Vector tool is a very old tool which we used to display the vector chain claimants.

I don't have any attribution for the source for this - it's possible that I wrote it, but
unlikely as I remember other people having copies back in the mid-late 90s.

In any case, this version was updated for RISC OS 4 to add the vector chain and the
number of vectors as collusion calls with the Kernel, so that it is a little more
authorised. It won't work on anything but a RISC OS Classic system, as it relies on the
format of the vector chain.
